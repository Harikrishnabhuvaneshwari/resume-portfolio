# Technical Resume Portfolio

A lightweight, responsive portfolio website template that mirrors resume content in a technical format.

## Files

- `index.html` - content structure (hero, experience, projects, skills, education, contact)
- `styles.css` - visual theme, responsive layout, and section styling
- `script.js` - reveal animation and dynamic year

## Customize with your resume

1. Replace all placeholder text in `index.html` with your real details.
2. Update the project cards with real repositories and live links.
3. Add your real resume PDF as `resume.pdf` in this folder.
4. Edit the skills and experience bullets to include metrics and impact.

## Run locally

Open `index.html` in your browser directly, or use a static server:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.
