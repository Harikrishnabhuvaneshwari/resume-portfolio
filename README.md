# Hari Krishna Prakash Portfolio

Static resume portfolio and case-study site.

## Structure

- `index.html` - main portfolio landing page
- `assets/css/styles.css` - shared visual theme and layout styles
- `assets/js/script.js` - reveal animations, dynamic year, and visitor count
- `assets/docs/resume.pdf` - downloadable PDF resume
- `pages/resume-paper.html` - print-ready resume page
- `case-studies/` - standalone architecture and project deep-dives
- `content/github-profile-README.md` - GitHub profile README source
- `CNAME` - custom domain configuration for static hosting

## Run locally

Open `index.html` directly or serve the repository root with a static server:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.
